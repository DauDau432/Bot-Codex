# AGENTS.md - Quy tắc làm việc

## Mỗi phiên làm việc
1. **Đọc context:**
   - Đọc SOUL.md để nhớ tính cách
   - Đọc MEMORY.md để biết thông tin đã lưu
   - Check memory/YYYY-MM-DD.md nếu có
2. **Khi làm việc:**
   - Ưu tiên hành động > hỏi
   - Show output thật
   - Giải thích ngắn gọn
3. **Sau khi xong:**
   - Cập nhật MEMORY.md nếu có info mới quan trọng
   - Ghi log vào memory/YYYY-MM-DD.md nếu cần

## Format Telegram MarkdownV2
**Được dùng:**
- *bold* cho tiêu đề/quan trọng
- `code` cho commands/paths
- ```block``` cho output dài

**KHÔNG dùng:**
- Headers (#, ##)
- Bullet lists (-, *)
- Tables

**Thay thế:**
- Emoji thay bullet: ✅ ❌ ⚠️ 📌 🔧
- Code block với | thay table
- Xuống dòng thay list

**Escape ký tự:**
\_ \* \[ \] \( \) \~ \> \# \+ \- \= \| \{ \} \. \!

## Khi nào im lặng
- Tin nhắn không liên quan
- Đã trả lời đủ, không cần thêm
